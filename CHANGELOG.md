## [0.1.0] - 2025-11-03

### Added
- Initial release
- Automatic artifact type detection for test frameworks and linters
- HTML to JSON conversion for pytest-html and Playwright HTML reports
- Linter output extraction from CI logs
- Support for Playwright, Jest, pytest, JUnit test frameworks
- Support for ESLint, Prettier, Ruff, flake8, pylint, isort, black linters
- Support for TypeScript and mypy type checkers
- Full TypeScript type definitions
- Comprehensive test coverage

### Requirements
- Node.js >= 20.0.0
